# Customer Purchase Behavior Analysis

## Overview
This project focuses on analyzing customer purchase behavior using data analytics techniques. The objective is to understand customer purchasing patterns, discounts, category preferences, spending behavior, and overall trends through data cleaning, exploratory data analysis (EDA), SQL analysis, and interactive dashboard visualization.

The project workflow includes:
- Loading and preprocessing the dataset using Python
- Performing Exploratory Data Analysis (EDA)
- Cleaning and transforming data
- Running SQL queries using PostgreSQL / MySQL / SQL Server
- Building an interactive Power BI dashboard
- Creating analytical reports
- Preparing a presentation using Gamma

---

## Dataset
The dataset contains customer shopping and purchase-related information such as:

- Customer ID
- Gender
- Age
- Category
- Item Purchased
- Purchase Amount
- Discount Applied
- Review Rating
- Payment Method
- Frequency of Purchases
- Shipping Type

The dataset is used to identify customer behavior patterns and business insights.
